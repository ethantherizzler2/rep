Reaper Terminal

Reaper Terminal is a powerful console-based network & Windows control tool. Perfect for power users needing advanced system commands in one place.

Download Reaper Terminal

Available Commands

hostinfo
Displays system hostname, domain, and network details.
whoami
Shows the current logged-in user and their permissions.
osinfo
Prints OS version, build, and architecture info.
ping
Sends ICMP requests to check if a host is reachable.
portscan
Scans TCP ports on a target host to detect open services.
netstat
Lists active connections, routing tables, and listening ports.
publicip
Fetches and displays your public IP address.
ddos
Launches a stress-test with repeated requests on a target.
ifconfig
Shows local network interface configuration and IPs.
wifiprofiles
Lists saved Wi-Fi profiles and optionally reveals keys.
ssl-info
Retrieves SSL/TLS certificate details from a host.
dnslookup
Looks up DNS records for a domain (A, MX, TXT, etc.).
dnsexpand
Expands subdomains and DNS records for enumeration.
http-head
Fetches HTTP headers from a URL for quick server info.
