What is Reaper?

Reaper is a powerful suite of tools designed for users who need advanced control over restricted or locked-down Windows systems. Whether you're working on a school laptop or a corporate device with limitations, Reaper provides the extra capabilities Windows doesn't allow by default.

With features like a network manager, process control, command line access, and Windows bypassing tools, Reaper is perfect for power users. Built using C++, C#, and Python.
